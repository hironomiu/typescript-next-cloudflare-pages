# typescript-next-cloudflare-pages

Cloudflare Pages + microCMS

## Run

```
yarn run dev
```

## Install Memo

```
npx create-next-app . --ts
yarn add microcms-js-sdk
```

### Tailwind

`yarn`で実行すること

[公式：nextjs](https://tailwindcss.com/docs/guides/nextjs)

### Testing

注意：`yarn add -D`で実行すること

[Jest and React Testing Library](https://nextjs.org/docs/testing#jest-and-react-testing-library)

```
yarn add -D ts-jest
yarn add -D @types/jest
```
